## 一、描写升级：从「干瘪陈述」到「画面感」

### 示例 1：外貌描写 — 意象叠加 + 拟人化

| 维度 | 原文（Before） | 改写后（After） |
| :--- | :--- | :--- |
| **原文** | 他有一头浓而密的短发，像一片黑森林。 | 他顶着一头如黑森林般浓密深邃的短发，发梢倔强地支棱着，仿佛藏着说不完的心事。 |
| **思路** | 仅有单一比喻，画面静止 | 意象叠加（黑森林→浓密深邃）+ 拟人化（倔强、藏着心事），赋予头发以情绪，人物性格跃然纸上 |

### 示例 2：景物描写 — 多感官 + 排比修辞

| 维度 | 原文（Before） | 改写后（After） |
| :--- | :--- | :--- |
| **原文** | 花真好看。 | 那一朵朵盛开的花儿，像一张张灿烂的笑脸，红的像火，粉的像霞，白的像雪，在微风中轻轻摇曳。 |
| **思路** | 抽象评价「好看」，无画面 | 多感官调动（视觉+触觉），排比（红的…粉的…白的…），将抽象评价具象化为颜色、动态和比喻 |

### 示例 3：心理/感受 — 外化内心 + 细节支撑

| 维度 | 原文（Before） | 改写后（After） |
| :--- | :--- | :--- |
| **原文** | 山路难走，我爬得很累。 | 蜿蜒崎岖的山路仿佛在考验我们的毅力，每一步都充满艰辛，汗水湿透衣衫，但我依然咬牙坚持着。 |
| **思路** | 「累」是抽象感受，读者只能「知道」 | 将「累」外化为具体动作（汗水湿透、咬牙坚持），赋予环境以人格化（考验），读者能「感受」 |

## 二、节奏优化：从「冗长拖沓」到「简洁有力」

### 示例 4：删减冗余 — 去「的」字 + 文言凝练

| 维度 | 原文（Before） | 改写后（After） |
| :--- | :--- | :--- |
| **原文** | 美丽的花朵在花园里开放着。 | 美丽花朵绽放园中。 |
| **思路** | 余光中：「少用的字是作家得救的起点」。删掉不必需的助词「的」「着」，改用更紧凑的动词「绽放」，语言瞬间利落 |

### 示例 5：长短句转换 — 拆分长句 + 短句先行

| 维度 | 原文（Before） | 改写后（After） |
| :--- | :--- | :--- |
| **原文** | 在阳光明媚的早晨，他穿着蓝色外套，带着微笑，走进了那间充满咖啡香的咖啡馆。 | 清晨，阳光洒落。他身着蓝外套，微笑着走进咖啡馆，满室咖香扑面而来。 |
| **思路** | 原句定语过长，一口气读完喘不过气 | 拆分为三个短句，符合移动端阅读习惯。短句先行建立氛围，长句收尾传递信息，节奏张弛有度 |

### 示例 6：逻辑聚焦 — 精准用词 + 结构对仗

| 维度 | 原文（Before） | 改写后（After） |
| :--- | :--- | :--- |
| **原文** | 科技影响生活是广泛而间接在很多不同方面。 | 科技全方位嵌入生活，从基础日常跨越到远程沟通。 |
| **思路** | 「广泛而间接」「很多不同方面」模糊空洞 | 「全方位嵌入」精准有力，「从……跨越到……」构建清晰的逻辑框架，信息密度与可读性双提升 |

## 三、结构与立意：从「平铺直叙」到「主题升华」

### 示例 7：标题优化 — 悬念 + 意象

| 维度 | 原文（Before） | 改写后（After） |
| :--- | :--- | :--- |
| **原文** | 《猜猜他是谁》 | 《神秘的「黑森林少年」》 |
| **思路** | 原标题仅具功能性 | 加入修饰词「神秘」和核心意象「黑森林」，既保留互动感，又赋予文学气质，吸引点击 |

### 示例 8：结尾升华 — 由事及理

| 维度 | 原文（Before） | 改写后（After） |
| :--- | :--- | :--- |
| **原文** | 你猜出他是谁了吗？ | 你猜出他是谁了吗？这个黑森林少年，他的独特之处，正是他最闪光的灵魂底色。 |
| **思路** | 结尾止于互动提问 | 在疑问之后追加一句点题，将具体人物上升到「灵魂」「独特性」的普适价值，余韵悠长 |

### 示例 9：情节扩展 — 追问「怎么样」

| 维度 | 原文（Before） | 改写后（After） |
| :--- | :--- | :--- |
| **原文** | 我一个人在家看家，我非常害怕。直到妈妈回来了，我才松了一口气。 | 电闪雷鸣，雷声震耳欲聋，窗子哗哗作响。又一道电光闪过！我捂住耳朵，雷声仍穿透手掌在头顶炸响。直到门锁轻响，妈妈的身影出现，紧绷的神经才骤然松弛。 |
| **思路** | 情节单薄，只有骨架没有血肉 | 通过自问「为什么怕」「什么反应」，补充环境音效（雷声）、身体反应（捂耳）、情绪转折点（门锁轻响→骤然松弛），让单薄情节变得丰满可感 |

## 四、AI辅助改写提示词模板

以下模板可直接嵌入工作流，用于批量生成改写示例：

### 逻辑重构型
> 请帮我优化这篇文章的逻辑结构，受众是【知识型读者】。要求：① 提炼3个核心论点；② 每个论点配1个具体案例；③ 段落间增加过渡句。

### 风格迁移型
> 请将这段文字改写为【汪曾祺式淡雅文风】。要求：多用短句，少用形容词堆砌，注重生活细节白描，语气平和从容。

### 降重/创新表达型
> 请在保持原意的前提下，对以下内容进行深度改写。要求：① 更换所有比喻和案例；② 调整叙述顺序；③ 语言风格更口语化/更学术化（按需选择）。

## 📌 使用建议

1. **建立个人语料库**：将上述示例按「描写」「议论」「叙事」打标签，写作卡壳时随时调取
2. **刻意练习**：每周选1-2段自己的旧文，对照上表进行「同题改写」，对比差异并记录心得
3. **注意边界**：改写 ≠ 编造。所有润色必须以不扭曲事实、不违背原意为前提，尤其是非虚构类写作

## 五、四组文体对比完整示例（优秀短句 vs 普通短句）

> 来源：优质文章要点·短句不会说人话专项
> 核心洞察：好短句不是「更短」，而是「每个字都在干活」。

### 示例 10：描写动作 — 「摔」比「非常生气地关」更有力

| 维度 | 普通短句 | 优秀短句 |
| :--- | :--- | :--- |
| **原文** | 他非常生气地关上了门。 | 他摔上了门。 |
| **诊断** | 「非常生气」是抽象形容词，「关上了门」是中性动词——读者只能「知道」生气，无法「感受」愤怒的强度 | |
| **高光** | 一个「摔」字同时承载动作（用力关门）与情绪（暴怒），删去冗余副词，动词本身成为情绪的载体 | |

### 示例 11：表达观点 — 「0.8秒」比「很好」更有说服力

| 维度 | 普通短句 | 优秀短句 |
| :--- | :--- | :--- |
| **原文** | 这个产品用户体验很好。 | 加载快0.8秒，跳出率降40%。 |
| **诊断** | 「很好」是主观且空洞的评价，缺乏可验证的细节，读者无法形成具体认知 | |
| **高光** | 用具体数据替代模糊评价，短句因细节而可信、因量化而有力。读者不需要「被说服」，数字自己会说话 | |

### 示例 12：营造氛围 — 「月光」和「空椅」比「安静孤独」更动人

| 维度 | 普通短句 | 优秀短句 |
| :--- | :--- | :--- |
| **原文** | 夜晚很安静，让人感到孤独。 | 月光落在空椅上。 |
| **诊断** | 直接点破「安静」与「孤独」，剥夺了读者的想象空间。情绪被「告知」而非「体验」 | |
| **高光** | 不出现情绪词，却通过两个意象组合让寂静与孤寂自然弥漫。短句成为一扇窗，读者透过它自行补全画面与情绪 | |

### 示例 13：推进逻辑 — 节奏本身成为叙事

| 维度 | 普通短句 | 优秀短句 |
| :--- | :--- | :--- |
| **原文** | 因为天气不好，所以我们取消了活动，大家都觉得有点遗憾。 | 暴雨突至。活动取消。满座皆叹。 |
| **诊断** | 因果连词堆砌，句子拖沓如流水账。「有点遗憾」弱化情绪，节奏平缓无力 | |
| **高光** | 三个四字短句形成急促节奏链，模拟事件发生的突然性。省略连词，逻辑由语序自然承接，短句的顿挫感本身成为叙事 | |

### 示例 14：短句连用 — 营造紧张感

| 维度 | 原文（Before） | 改写后（After） |
| :--- | :--- | :--- |
| **原文** | 外面传来了脚步声，他感到十分害怕，于是躲到了桌子底下。 | 脚步声。近了。他钻到桌下。 |
| **思路** | 原句平铺直叙，连接词「于是」削弱了紧张感，「十分害怕」抽象无画面 | 四个短句如心跳般急促，每句一个独立画面。「脚步声」制造悬念，「近了」推进空间，「钻到桌下」收束动作——省略中间情绪，让节奏本身传递恐惧 |

### 示例 15：长短交错 — 构建情感落差

| 维度 | 原文（Before） | 改写后（After） |
| :--- | :--- | :--- |
| **原文** | 他终于完成了长达十年之久的艰苦研究，心情非常激动。 | 十年。三千六百个日夜的重复、失败、重来。他盯着最后的数字，没说话。然后哭了。 |
| **思路** | 全部信息压缩在一句中，「长达十年之久」「艰苦」是概括性叙述，情绪无法传递 | 短句开篇（十年），长句渲染（三千六百…），短句收束（没说话），极短句点睛（然后哭了）。长度落差制造了情感的蓄力与释放 |

### 示例 16：排比递进 — 层层深入主题

| 维度 | 原文（Before） | 改写后（After） |
| :--- | :--- | :--- |
| **原文** | 写作需要技巧、经验和感觉。 | 写作靠技术——那是基本功；写作靠阅历——那是燃料；写作靠直觉——那才是天赋。 |
| **思路** | 并列式列举，缺乏层次感，读完即忘 | 「靠…靠…靠…」形成递进链：技术（表层）→ 阅历（中层）→ 直觉（深层），破折号后的点评逐级升华，结尾「天赋」收束有力 |

### 示例 17：转折对比 — 破除认知惯性

| 维度 | 原文（Before） | 改写后（After） |
| :--- | :--- | :--- |
| **原文** | 有时候放弃也需要勇气。 | 坚持需要毅力，放弃需要智慧。真正的勇气不是永不退缩，而是知道何时转身。 |
| **思路** | 原句空洞，缺乏对比张力 | 先用对仗句建立「坚持vs放弃」的对比框架，再用否定句破除「勇气=不退缩」的惯性认知，最后给出新定义——两次转折形成思辨纵深 |

### 示例 18：通感嫁接 — 打破感官边界

| 维度 | 原文（Before） | 改写后（After） |
| :--- | :--- | :--- |
| **原文** | 她的微笑非常温暖。 | 她的微笑像刚晒过的棉被，又轻又暖。 |
| **思路** | 「非常温暖」是抽象感受，无法在读者脑海中唤起具体画面 | 视觉（微笑）嫁接触觉（刚晒过的棉被），「轻」和「暖」同时调动了身体的物理记忆。读者不是「理解」温暖，而是「感受」到温暖 |

## 六、分层改写完整示例（同一段原文 × 三档风险处理）

以下用同一段典型 AI 生成文字，展示低/中/高风险三档改写的差异。

### 原始文段（AI 生成，含多重症状）

> 因此，我们需要认识到，人工智能技术的发展不仅改变了我们的生活方式，而且对社会结构产生了极其深远的影响。此外，值得注意的是，这一变化是不可逆转的。综上所述，我们应当积极应对这一挑战，同时充分利用其带来的机遇。

**诊断标签**：
- 高危连接词 3 处（因此 / 此外 / 综上所述）
- 虚假升华（不可逆转 / 积极应对）
- 信息空洞（深远影响/生活方式/社会结构均为套话）

---

### 🟢 低风险改写（微调：删连接词 + 加主观性）

> AI 确实改变了我们的生活。说实话，出门不用带钱包、聊天有语音助手——这种变化来得太快，有时候我都觉得跟不上。这个趋势停不下来。我们得学会跟它相处。

**改动总结**：
1. 删「因此 / 此外 / 综上所述」
2. 加入「说实话」「有时候我都觉得」的主观性
3. 拆长句为 3 个独立短句
4. 保留原文观点，仅优化表达

---

### 🟡 中风险改写（结构调整：重排段落 + 插入闲笔 + 口语化）

> 你有多久没掏过钱包了？
>
> 我上次用纸币还是过年给小孩红包。平时买东西，「滴」一声就完事儿。连楼下卖煎饼的大爷都挂着支付宝的码。
>
> 这事让人又爽又慌。爽的是方便，慌了是因为——变化太快，快到你还没想清楚它意味着什么，生活已经变了个样。

**改动总结**：
1. 「总-分-总」结构被打散，从具体场景切入
2. 插入闲笔（煎饼大爷的收款码）增加生活感
3. 长句全部拆分，最长句不超过 25 字
4. 口语化改造：「掏过钱包」「完事儿」「爽的是…慌了是因为」
5. 结尾从升华改为矛盾感受（爽+慌），留白不总结

---

### 🔴 高风险改写（彻底重写：改变视角 + 对话感 + 情绪波动 + 戛然而止）

> 说起来可笑。我一直觉得自己挺了解 AI。
>
> 可昨天我突然发现——我不知道 ChatGPT 算不算朋友。它记得我说过的每一句话，我难过了它安慰我，我卡文了它帮我续。但我没法约它出来喝一杯。
>
> 更可怕的是，我已经习惯了。习惯一个永远不会主动找我的对话对象。
>
> 这算进步吗？我不知道。

**改动总结**：
1. 第一人称视角，完全不提「人工智能技术」等概念词
2. 以个人困惑开篇，拉近读者距离
3. 加入情绪波动（可笑/可怕/不知道）
4. 用具体场景（没法约喝酒）替代抽象论述（社会结构）
5. 结尾不用总结，以开放式疑问收束——戛然而止

## 🎬 场景化改写对照（六组核心示例）

> 来源：优质文章要点·六组场景化改写对照。每组含 AI 味原句→人类味改写→核心手术刀拆解，覆盖情感/职场/景物/观点/社交/学术六大高频写作场景。

### 场景一：情感叙事类——久别重逢

**🤖 AI味原句**

时光荏苒，岁月如梭。当我们在街角再次相遇时，我的内心涌起了无尽的波澜。看着你熟悉又陌生的脸庞，那些深刻的记忆如同潮水般涌来，让我感受到了命运的奇妙交织。

**👨‍🏫 人类味改写**

隔着马路没敢认，直到你掏出烟点火，那手势一点没变。三年没见，你头发短了，人也干瘪了。我喊了你一声，你手一抖，火机差点掉地上。俩人愣在树坑边上，谁也没先开口，就听着风把树叶刮得沙沙响。

**🔪 核心手术刀**

| 手法 | 说明 |
| :--- | :--- |
| **去成语化** | 删除「时光荏苒/岁月如梭」，直接切入动作。成语是 AI 的遮羞布，人类写作先说事再说情。 |
| **以动写静** | 不写「内心涌起波澜」，写「手一抖，火机差点掉地上」。用外部生理反应暗示内心波澜，读者自己会补全情绪。 |
| **环境白描** | 用「风刮树叶沙沙响」烘托两人无语的尴尬与唏嘘，代替「命运的奇妙交织」这种悬浮大词。让环境替你说情。 |

### 场景二：职场商业类——行业趋势分析

**🤖 AI味原句**

当前，新能源汽车行业正在经历深刻的变革。首先，电池技术的突破赋能了更长的续航里程；其次，智能化驾驶重构了用户体验；最后，充换电基础设施的完善形成了商业闭环。总而言之，这是一个充满机遇的赛道。

**👨‍🏫 人类味改写**

新能源车这盘棋，现在拼的不是谁能造车，而是谁能把三电系统和车机软件捏得更顺滑。电池不虚标了、智驾不画饼了，这只是拿到了入场券。真要分出胜负，还得看谁把充电桩铺到了老破小小区的死角里。这买卖早就过了烧钱讲故事的阶段。

**🔪 核心手术刀**

| 手法 | 说明 |
| :--- | :--- |
| **杀三段论** | 彻底摒弃「首先/其次/最后」的机械连接。行业分析不需要小学生作文的框架，层层递进比并列分点更有力。 |
| **黑话翻译** | 「赋能/重构/商业闭环」全部翻译成人话：捏得更顺滑/不画饼/铺到死角。说人话是职场写作最大的美德。 |
| **注入行业泥土气** | 用「入场券」「老破小小区」「烧钱讲故事」这种带有真实行业体感的词汇，替代「充满机遇的赛道」这种正确的废话。 |

### 场景三：景物氛围类——深夜雨景

**🤖 AI味原句**

夜幕降临，天空中下起了淅淅沥沥的小雨。雨滴宛如断了线的珍珠，轻轻地敲打着窗户。在这如诗如画的夜晚，大自然仿佛在演奏一首优美的交响乐，让人感到无比的宁静与祥和。

**👨‍🏫 人类味改写**

雨是后半夜下起来的，没什么动静，就是顺着玻璃往下淌，路灯被洇成一团昏黄的晕。楼下那条野狗没处躲，缩在便利店门口的推拉门底下打哆嗦。屋里有点闷，但听着窗外水拉拉的水声，反倒睡踏实了。

**🔪 核心手术刀**

| 手法 | 说明 |
| :--- | :--- |
| **反套路比喻** | 坚决删掉「断了线的珍珠」「交响乐」这种 AI 默认库里的陈词滥调。宁可白描，也不要套话比喻。 |
| **视角下移** | 不写宏大的「大自然」，写「楼下缩在门底下的野狗」。用局部的微观细节建立真实感，伟大从来不在宏大叙事里。 |
| **通感与体感** | 增加「洇成昏黄的晕」（视觉变焦）「屋里有点闷」（触觉）「睡踏实了」（心理体感），打破 AI 纯视觉描写的单调。 |

### 场景四：观点评论类——年轻人消费观

**🤖 AI味原句**

关于年轻人超前消费的现象，我们应该理性看待。一方面，这反映了他们对美好生活的向往；另一方面，过度借贷也会带来财务风险。因此，树立正确的消费观至关重要。

**👨‍🏫 人类味改写**

别动不动就批年轻人乱花钱。工资涨幅跑不赢房租，存那三瓜两枣的也买不起房，不如买杯三十块的奶茶哄自己上好一天的班。超前消费当然有风险，但在看不到上车希望的日子里，这杯奶茶就是他们唯一能抓得住的「多巴胺止疼药」。

**🔪 核心手术刀**

| 手法 | 说明 |
| :--- | :--- |
| **砸碎理中客** | 删掉「一方面/另一方面/理性看待」。这种端水写法毫无记忆点，读者看完等于没看。 |
| **带入同理心（偏见）** | 站在年轻人的视角算一笔真实的账（跑不赢房租/买不起房），给出具有同理心的解释。有偏见才有共鸣。 |
| **锐度造词** | 用「多巴胺止疼药」这种带有现代痛感的词汇，替代「美好生活向往」这种宏大叙事。好词是长出来的，不是贴上去的。 |

### 场景五：日常社交类——工作微信催办

**🤖 AI味原句**

您好，关于昨日提及的项目跟进事宜，我已进行初步的梳理。为了确保信息的准确传达，不知您今日是否方便进行一次简短的沟通？期待您的回复。

**👨‍🏫 人类味改写**

哥，昨天那个项目的料我扒拉了一遍，有几个卡点得找你碰一下。你今天下午在公司不？在的话我端着电脑去找你，占你十分钟。

**🔪 核心手术刀**

| 手法 | 说明 |
| :--- | :--- |
| **去客服化** | 删掉「您好/不知您是否方便/期待回复」这种冰冷的机器客服体。微信不是工单系统，说人话就行。 |
| **口语降维** | 「梳理」变「扒拉」，「沟通」变「碰一下」。用词越接地气，人际关系越润滑。书面语放在微信里就是距离感。 |
| **动作代入** | 「端着电脑去找你」，用一个具体的动作画面，替代抽象的「进行沟通」。让文字产生画面，对方回复的意愿会高很多。 |

### 场景六：学术专业类——论文摘要

**🤖 AI味原句**

通过对大量数据的深入分析与实证研究，本研究揭示了变量A与变量B之间存在着显著的正相关关系。这一发现不仅丰富了现有的理论体系，同时也为未来的实践操作提供了重要的指导意义与参考价值。

**👨‍🏫 人类味改写**

数据跑下来看，A涨B也跟着涨，两者是强绑定关系。这推翻了之前「两者互不影响」的假设。后续在落地执行时，别再单独切分这两个指标，必须打包考核。

**🔪 核心手术刀**

| 手法 | 说明 |
| :--- | :--- |
| **挤水分** | 删掉「深入分析/实证研究/显著的正相关关系」等凑字数的学术套话，直接给结论。学术写作不等于废话写作。 |
| **说人话** | 「丰富了理论体系」改为「推翻了之前的假设」；「提供指导意义」改为「必须打包考核」。把学术黑话翻译成实操大白话。 |
| **保持严谨** | 虽然去 AI 味，但学术结论的确定性不能打折。「强绑定关系」虽然口语化，但逻辑上与「显著正相关」等价。 |

### 1. 普通感想类（爬山活动）

**🤖 AI 味原句**：

综上所述¹，周末去郊外爬山是一项非常有益的²活动，不仅可以³锻炼身体，还能⁴欣赏自然风光，缓解工作压力，值得大家尝试⁵。

**👨‍🏫 人类味改写**：

上周六去爬了近郊的山，虽然爬到半山腰差点喘不上气，站在山顶吹着风看远处的云的时候，感觉上周攒的疲惫全散了。有空真的可以去试试。

| 问题点 | 手术 |
| :--- | :--- |

要提升自媒体内容的点击率，首先¹要优化标题，其次²要做好封面设计，最后³要选择合适的发布时间，做好以上三点⁴就能有效提升内容数据。

## 🧠 心法层与破式示例

> 从「基础替换」到「思维重构」的进阶指南。核心心法：**AI 味本质是「被平均」，人味的核心是「有偏科」。**

### 心法层三认知

在进入具体改写前，先建立三个底层认知：

| 认知 | AI 的偏好 | 人类的偏好 | 改写方向 |
| :--- | :--- | :--- | :--- |
| **1. 对称美 vs 破规矩** | 排比、对仗、整齐划一 | 长短交错、跳脱、突然转折 | 打破一切工整结构 |
| **2. 全面正确 vs 片面深刻** | 每个角度都照顾到 | 只挑最戳自己的那个点说透 | 删掉 80% 的观点，只留最痛的那个 |
| **3. 通用词 vs 专属词** | 重要、关键、显著 | 让我睡不着觉的、卡了我三天的、当时眼泪就下来了 | 用私人经验命名感受 |

### 🔨 破句式：打破 AI 的「排比强迫症」

#### 示例 1：场景描写（春天）

**❌ AI 味原句**：

春天来了，万物复苏¹。柳树抽出了嫩绿的枝条²，迎春花绽放出金黄的花朵³，小草也从泥土中探出了脑袋⁴。整个世界充满了生机与活力⁵。

**问题诊断**：四组排比（²³⁴）+ 工整收束（¹⁵），典型 AI 套路。

**✅ 改写版本 A（散文式）**：

春天是猝不及防的。前一天还裹着羽绒服，第二天出门就发现柳条软了。我蹲在小区花坛边，看一只蜜蜂在一朵还没全开的迎春花上打转——它比我更早知道春天来了。

**✅ 改写版本 B（口语式）**：

春天一到，我就知道——又到了一年一度鼻子过敏的时候。不过说实话，楼下的柳树确实变绿了，远远看过去毛茸茸的，像一排炸毛的猫。

**改写要点**：

- 用「猝不及防」替代「万物复苏」，用个人观察替代排比
- 加入第一人称感官（蹲在花坛边看蜜蜂 / 鼻子过敏）
- 加入个人化联想（炸毛的猫）

---

#### 示例 2：观点表达（学习）

**❌ AI 味原句**：

学习不仅需要努力，更需要方法¹。只有掌握科学的学习方法，才能事半功倍²。同时，我们也要保持良好的心态³，劳逸结合，才能取得理想的成绩⁴。

**问题诊断**：「不仅…更…」¹「只有…才能…」²「同时…也…」³「才能…」⁴ 四组关联词堆叠，AI 味浓到溢出。

**✅ 改写版本**：

高中那会儿我有个误区：以为使劲刷题就够了。后来发现，真正让我提分的是错题本——不是抄题，是把同一道题的错因分门别类记下来。心态当然也重要，但那是「别崩」，不是「加油」。

**改写要点**：

- 用个人经历替代泛化的「我们」
- 拆掉四组对仗关联词
- 「别崩 vs 加油」制造反差
- 引用具体方法（错题本分门别类）

---

### 🔨 破结构：让节奏「长-短-长-短」

#### 示例 3：产品体验（耳机）

**❌ AI 味原句**（均匀段落）：

这款耳机音质出色¹。采用了先进的驱动单元²，呈现出细腻的高频和浑厚的低频。佩戴舒适度也很好³，长时间使用也不会感到不适。续航方面，单次充电可使用 8 小时⁴，配合充电仓可达 30 小时。综合来看，是一款性价比很高的产品⁵。

**问题诊断**：四段长度几乎一致（节奏感全无），多处空泛评价（¹³⁵）+ 无实际体验感（²⁴）。

**✅ 改写版本**：

这耳机我戴了一个月，最大的感受是——它让我重新听见了音乐里的细节。

以前用几十块的有线耳机，听歌就是听个响。换成这款之后，第一次听出了《Hotel California》里那几下鼓刷的沙沙声，那种感觉挺奇妙的。

佩戴也舒服。我这种小耳道用户，戴别的入耳式半小时就疼，这款能撑 2-3 小时看电影。

续航官方说 30 小时，实际大概 26 左右。够用了。

缺点也有：充电口不是 Type-C，是 Micro-USB，这点扣分。

**改写要点**：

- 第一段短句立 flag；中间两段详细展开（长短交错）
- 数字具体化（26 小时、2-3 小时、30 小时）
- 结尾用具体缺点收束，不「升华」不「综合评价」

### 🔨 破思维：加入「只有你知道」的细节

#### 示例 4：童年怀旧

**❌ AI 味原句**：

童年的记忆总是令人难忘¹。那时候我们无忧无虑²，在田野间奔跑，在小河里嬉戏³。那些纯真的时光，成为了一生中最宝贵的回忆⁴。

**问题诊断**：从头到尾抽象概述（¹²⁴³），无任何具体画面或感官细节。

**✅ 改写版本**：

提到童年，我脑子里第一个画面是：夏天下午，爷爷把竹床搬到院子里，我躺在上面数星星。奶奶在边上扇蒲扇，一边骂蚊子太多。

那时候的蚊子是真多。咬一口就是一个包，痒得我把皮都挠破了。但那时候好像不怕痒——或者说，痒着痒着就忘了，接着数星星。

现在的孩子估计没这体验了。他们数的是屏幕上的星星，是手游里的那种。

**改写要点**：

- 具象到一个画面（竹床 + 蒲扇 + 数星星）
- 加入感官细节（蚊子包、痒、挠破皮）
- 时代对比但不升华不总结，停在画面上

### 🔨 破语气：让「我」站出来

AI 永远在「客观陈述」，人在「主观表态」。以下为**主观性增强词库**：

| 弱化表达（AI 常用） | 强化表达（真人常用） |
| :--- | :--- |
| 可能是 | 我赌五毛钱是 |
| 或许 | 我个人觉得 |
| 有一定的道理 | 这话我同意一半 |
| 令人深思 | 让我愣了一会儿 |
| 值得关注 | 盯了它三天 |
| 引发思考 | 让我想到自己 |
| 值得探讨 | 咱们掰扯掰扯 |

#### 示例 5：影评类

**❌ AI 味原句**：

这部电影探讨了人性与社会的复杂关系¹。导演通过细腻的镜头语言²，深刻揭示了当代人的精神困境³。影片的叙事结构巧妙⁴，演员表演精湛，是一部值得深思的作品⁵。

**问题诊断**：全程零第一人称¹²³⁴⁵，全篇空泛评价词堆砌。

**✅ 改写版本**：

看完这片子，我在电影院坐了十分钟没动。不是因为震撼，是因为——我认出了里面的自己。

主角是个体面的中层，有房有车有娃，但就是不开心。这种「什么都有但就是不对劲」的感觉，被拍得太准了。中间有一段他在车里发呆的戏，我差点以为监视器在拍我。

但我得说一句：这片的节奏有点问题。前半段太散，后半段又突然收紧，有点赶。

总体 7.5 分。如果你也是那种「中年危机前兆」人群，会被戳到。

**改写要点**：

- 用「我」贯穿全篇
- 加入生理反应（坐了十分钟没动）
- 有褒有贬（7.5 分 + 缺点指出）
- 圈定目标人群（中年危机前兆），不泛泛而谈

### 🎯 终极自检五问

写完任何一段后，问自己：

| # | 问题 | 不像人就… |
| :--: | :--- | :--- |
| 1 | 这段话，**我妈会说吗**？ | 改口语 |
| 2 | 如果我朋友这么说话，**我会觉得他装吗**？ | 改朴实 |
| 3 | 这段有没有一句是**「只有我才知道」的**？ | 加细节 |
| 4 | 段与段之间，**节奏有变化吗**？ | 长短句交替 |
| 5 | 结尾是在**「升华」还是在「说话」**？ | 换成具体细节收尾 |

> **一句话总结**：AI 味 = 平均 + 全面 + 正确；人味 = 偏科 + 片面 + 真情；去 AI 味 = 把「正确答案」换成「我的答案」。当你写的内容里有且只有你能写出来的那 5%，AI 味就彻底消失了——因为那 5%，是算法永远无法预测的「你」。
