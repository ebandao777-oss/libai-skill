# 中文 AI 味禁用词与句式表

---

## 一、最毒禁用句式（出现即修，最高优先级）
| 毒级 | 句式 | 错误例 | 修法 |
|------|------|--------|------|
| ★★★★★ | "不是A，而是B" / "不是A，不是B，而是C" | "他不是冷漠，而是绝望" | 直接写 B 或用更自然的表达 |
| ★★★★ | "，带着……" 万能状语 | "他笑了一下，带着一丝不易察觉的嘲讽" | 拆短句或换动作 |
| ★★★★ | "声音不大，却带着一种……的力量" | "她声音不大，却带着不容置疑的力量" | 直接写声音特征或动作 |
| ★★★★ | "他/她知道……" | "他知道这一切都来不及了" | 用行为展示认知 |
| ★★★ | "仿佛/犹如/宛若……一般" | "仿佛能穿透一切一般" | 删掉或白描 |
| ★★★ | "眼中闪过一丝……" / "嘴角勾起一抹……" | "眼中闪过一丝悲伤" | 他垂下眼 / 他笑了一下，没到眼底 |
| ★★★ | "心中涌起一股……" / "心头一震" | "心中涌起一股暖流" | 用身体反应 |
| ★★ | 章末预告 "他不知道的是……" | "他不知道的是，更大的风暴即将来临" | 用具体钩子物件/事件收束，避免空泛预告 |

## 四、中文 AI 高频连接词/填充词
### 连接词（过度使用时标记）
此外、同时、值得注意的是、需要指出的是、不难发现、基于以上分析、综上所述、总而言之、首先/其次/最后、一方面/另一方面、不可否认、毋庸置疑、随着...的发展、在...的背景下、从某种意义上说、在当今社会、至关重要、深入探讨、进一步来说、换言之、由此可见

- "不得不说"
- "坦率地说"
- "客观来讲"
- "众所周知"
- "毫无疑问"

- "影响深远"
- "问题的根源在于结构性因素"
- "其重要性不言而喻"
- "这引发了广泛的讨论"

## 五、比喻分类（默认全删）
| 比喻类别 | 例 |
|---------|----|
| 动物类 | "像一头被抛弃的野狗" |
| 物品/现象类 | "像一把刀" "脸色惨白得像这漫天的雪" |
| 状态类（陈词滥调） | "梨花带雨" "如沐春风" |
| 抽象类 | "像命运的齿轮" "像上辈子的尘埃" |
| 假设类 | "力道大得像是要把骨头捏碎" |

**处理原则**：把比喻改为直接描述／动词／名词／作用／结果／事实。例："脸色惨白得像这漫天的雪" → "脸色惨白"。

## 六、替换策略速查
| 原文类型 | 替换方法 | 示例 |
|----------|----------|------|
| 抽象情绪词 | 具体动作 | "紧张" → "手在抖" |
| "感到XX" | 外化表现 | "感到愤怒" → "攥紧拳头" |
| 形容词堆砌 | 白描手法 | "美丽动人的笑容" → "她笑了" |
| 书面表达 | 口语化 | "不容置疑" → "就是" |
| 解释性描写 | 留白 | "他因为害怕而..." → "他退后一步" |
| 连续排比 | 保留最强一条 | 3 句排比留 1 句 |
| 总结升华句 | 直接删除 | "这一刻，她终于明白了..." → 删 |
| "不是A，而是B" | 直接写 B | "他不是冷漠，而是绝望" → 直接写 B |
| 多余修饰 | 删 | "白色的药片" → "药片"；"手里那截链子" → "链子" |

## 八、高频踩雷词（来自 khazix-writer，出现即修）

| 禁用表达 | 问题 | 替换建议 |
|----------|------|----------|
| "说白了" | AI特别爱用，一出现立刻暴露 | 删掉，直接说 |
| "意味着什么？" | AI标志性句式 | 删掉，或换成具体追问 |

| "换句话说" | 太书面 | 删掉，直接说 |
| "不可否认" | 套话 | 删掉 |
| "综上所述" / "总的来说" | 套话 | 删掉，用动作/场景收尾 |
| "首先...其次...最后" | 三段式 | 打散，用隐性连接 |
| "值得注意的是" / "不难发现" | 套话 | 删掉，直接说内容 |
| "让我们来看看" / "接下来让我们" | 套话 | 删掉，直接进入内容 |
| "在当今...的时代" | 教科书开头 | 换成具体事件切入 |
| "随着技术的不断进步" | 空话开头 | 换成具体事件切入 |
| "比如有一次..." | 假设性例子 | 换成"就像我今天正在搞的..."真实场景 |

## 十二、空洞动词黑名单（去除抽象万能动词）
> 来源：优质文章要点·高频AI词汇黑名单

| 禁用词 | 风险 | 替换建议 |
|--------|------|----------|
| 进行 + 名词 | 🔴 高 | 直接动词（「进行讨论」→「讨论」），90%情况直接删「进行」 |
| 实现 / 达成 / 完成 | 🟡 中 | 做到、搞定、落地、做出 |
| 提供 / 给予 / 赋予 | 🟡 中 | 给、送、帮、支持，或补全主语 |
| 旨在 / 致力于 / 意在 | 🟡 中 | 为了、想、要、图 |
| 导致 / 造成 / 引发 | 🟡 中 | 让、使（中性）；惹出、引来（负面） |
| 利用 / 运用 | 🟢 低 | 用、靠、借、拿 |
| 涵盖 / 包含 / 涉及 | 🟢 低 | 有、包括、覆盖了哪些 |
| 彰显了 / 凸显了 / 体现了 | 🟡 中 | 用具体动作替换抽象宣告 |

## 十五、翻译腔禁用句式（拆解英式语法结构）
> 来源：优质文章要点·翻译腔与冗长句式

| 禁用句式 | 风险 | 替换建议 |
|--------|------|----------|
| 「的」字超长定语（3个「的」以上连缀） | 🔴 高 | 从「的」字处切断，拆为2-3个独立短句。例：「这是一个充满了挑战与机遇的、需要我们全力以赴的、具有历史意义的时刻」→「这时刻有挑战也有机遇。历史会记住它，我们必须全力以赴。」 |
| 对于...而言 / 就...来说... | 🟡 中 | 删掉介词结构，直接说主语。「对于年轻人而言」→「年轻人」 |
| 在...的过程中 / 在...方面 | 🟡 中 | 直接删除，说出核心动作。「在沟通的过程中」→「沟通时」；「在学习方面」→「学习上」 |
| 被...所...（被动句滥用） | 🟡 中 | 改为主动句。「被广泛使用」→「大家都在用」；「被世人所熟知」→「大家都知道」 |
| 进行了...的讨论 / 给出了...的评价 | 🔴 高 | 动词直接跟上宾语。「进行了讨论」→「讨论了」；「给出了评价」→「评价道」 |
| 通过...的方式 / 以...的形式 | 🟡 中 | 删除冗余框架。「通过写信的方式」→「写信」；「以会议的形式」→「开会」 |
| 具有...的意义 / 起到...的作用 | 🟡 中 | 用具体动作替代。「具有重要意义」→「帮上了忙 / 让...成为可能」 |

**翻译腔三连杀口诀**：①找到「的」字超长定语 → ②砍掉「对/就/在/被」开头的介词框 → ③把所有「进行了/给出了/具有了」还原为实义动词。

## 十六、伪情感与客服腔禁用词（拆除假共情）
> 来源：优质文章要点·伪情感与客服腔

| 禁用词/句式 | 风险 | 替换建议 |
|--------|------|----------|
| 内心涌起波澜 / 心中五味杂陈 | 🔴 高 | 用生理反应或动作代替：「手心直冒汗」「嗓子眼发紧」「半天说不出话」 |
| 如同潮水般涌来 / 涌上心头 | 🔴 高 | 具象化感受：「突然像被人抽干了力气」「一股无名火从脚底板窜上来」 |
| 您好，关于您提及的... | 🔴 高 | 拟人化口语：「哥，你昨天说的那事」「嘿，上次聊到的那个项目」 |
| 不知您是否方便... | 🟡 中 | 给出具体时间选项：「今天下午3点通个话行吗？」「周二上午有空不？」 |
| 大自然的魅力 / 岁月的沉淀 | 🔴 高 | 写出具体的刺痛、冷暖、气味：「山风灌进领口，凉得人一激灵」「这本日记的纸页已经发黄，一翻就簌簌掉渣」 |
| 期待您的回复 / 感谢您的理解 | 🟡 中 | 自然收尾：「回头聊」「随时找我都行」「辛苦你跑一趟」 |
| 希望以上内容对您有帮助 | 🔴 高 | 直接删除，不要让文本以客服腔结尾 |
| 非常抱歉给您带来了不好的体验 | 🔴 高 | 改为人话：「这次是我们没弄好」「出了这档子事，确实对不住」 |

**客服腔判定红线**：文本中出现「您好」「不知您是否」「期待您的回复」「感谢您的理解」「希望以上内容」任一句式，即标记为客服腔。✅ 例外：正式商务邮件、客服工单等场景可酌情保留。

## 十七、高频禁用词补充（五组精细化屏蔽）
> 来源：优质文章要点·高频禁用词

### 组一：冗余逻辑衔接词
> 真人表达不会密集使用规整连接词，过度使用暴露AI生成特征。

| 禁用词 | 风险 | 替换建议 |
|--------|------|----------|
| 首先、其次、再次、最后 | 🔴 高 | 删除序数，自然递进 |
| 综上、综上所述、不难看出、不难发现 | 🔴 高 | 删掉，直接给结论 |
| 值得一提的是、需要强调的是、客观来说、公平来讲 | 🟡 中 | 直接陈述事实，不铺垫 |
| 从某种意义上来说、换句话来说、也就是说 | 🟡 中 | 删除冗余过渡，除非确实需要重新解释 |
| 有一说一、划重点 | 🟢 低 | 可偶尔口语化使用，但不要堆砌 |

### 组二：生硬共情/滥用热词
> AI硬加网络热词来显得接地气，反而暴露机器本质。

| 禁用词 | 风险 | 替换建议 |
|--------|------|----------|
| 家人们、宝子 | 🔴 高 | 仅在个人真实风格中使用，不确定就别加 |
| 谁懂啊、我真的哭死、绝绝子、yyds | 🔴 高 | 网络热词只用在你真的有对应情绪的场景 |
| 狠狠共情了、一整个爱住、破防了 | 🔴 高 | 同上，脱离情绪支撑的滥用非常假 |
| 咱就是说、一整个大动作 | 🟡 中 | 偶尔口语使用，不要密集出现 |

### 组三：模糊甩锅/绝对化表述
> AI要么打太极要么乱下结论，两种极端都暴露机器属性。

| 禁用词 | 风险 | 替换建议 |
|--------|------|----------|
| 一般来说、普遍认为、大概率、可能、也许 | 🟡 中 | 「一般来说」可换为「我见过的」「按常理」 |
| 据不完全统计、有研究表明、相关数据显示、专家指出 | 🔴 高 | 替换为具体来源：「2024年国家统计局发布的XX数据显示」 |
| 完全可以、绝对不会、百分百、毋庸置疑 | 🔴 高 | 加不确定感：「基本可以确定」「目前来看」「大概率没错」 |
| 底层逻辑、认知差、信息差 | 🟡 中 | 用大白话解释：「根子上的原因」「知道的人少」「知道的人多但做到的少」 |

### 组四：模板化评价/悬浮黑话
> 套任何内容都能用，无具体指向性，是AI最喜欢的凑字数手段。

| 禁用词 | 风险 | 替换建议 |
|--------|------|----------|
| 节奏紧凑、剧情饱满、人物立体、立意深刻 | 🟡 中 | 具体描述为什么好：「第三集那个反转，看了三遍才捋清楚」 |
| 发人深省、瑕不掩瑜、高开低走、口碑爆棚、年度最佳 | 🔴 高 | 用个人化评价替代：「看到最后十分钟，我坐沙发上愣了半天」 |
| 对齐、拉通、颗粒度、闭环、赋能、抓手 | 🔴 高 | 翻译成人话（参见第十四章） |
| 势能、调性、迭代、下沉 | 🟡 中 | 「势头/气质/升级/往下走」 |

### 组五：AI专属客套词
> 这些是AI结束一段回答时的「签名档」，出现即暴露。

| 禁用词 | 风险 | 替换建议 |
|--------|------|----------|
| 希望我的回答能帮到你 | 🔴 高 | 直接删除 |
| 如果你有其他问题欢迎随时提问 | 🔴 高 | 直接删除 |
| 非常抱歉给您带来了不好的体验 | 🔴 高 | 改为「这次没做好，对不住」 |
| 感谢您的理解与支持 | 🔴 高 | 直接删除，或改为「谢了」「辛苦你」 |

**执行口诀**：不是上述内容完全不能用，而是要避免集中扎堆使用——「哦对了」「话说回来」「害」这类口语化衔接词反而更真实。

## 十八、英式翻译腔禁用词
AI 语料库含大量英文翻译材料，导致中文输出带「英语从句结构」翻译腔。以下 10 个高频词来自真实 AI 输出统计，任何一个出现在中文文本中都应立即处理。

| 词汇 | 典型语境 | 替代建议 | 风险 |
| :--- | :--- | :--- | :--: |
| **delve** | delve into | 深入、仔细看看 | 🔴 高 |
| **realm** | in the realm of | 在……方面 | 🔴 高 |
| **underscore** | underscores the importance | 体现出、说明 | 🔴 高 |
| **meticulous** | meticulous analysis | 仔细、认真 | 🔴 高 |
| **robust** | robust solution | 靠谱、稳的 | 🟡 中 |
| **leverage** | leverage the power | 借助、用上 | 🔴 高 |
| **tapestry** | rich tapestry | 画面、图景 | 🔴 高 |
| **pivotal** | pivotal role | 关键、重要 | 🟡 中 |
| **beacon** | beacon of hope | 代表、象征 | 🔴 高 |
| **testament** | testament to | 证明、说明 | 🟡 中 |

> 这些词在英文写作中完全正常，但在中文 AI 输出中极度扎堆，已被主流 AIGC 检测系统列为特征词。全文搜索 Ctrl+F 扫描一遍，发现一个替换一个。

## 十九、比喻通感套路句式
AI 高频使用的陈词滥调比喻和煽情句式，本质是英译中或网文模板化表达，中文真人写作极少高频使用。

| 类别 | 禁用句式模板 | 典型例句 | 人类替代 |
| :--- | :--- | :--- | :--- |
| 🔴 眼睛当刀 | 目光/眼神/视线像……的刀，狠狠扎在……心上 | 她眼神像淬了冰的刀，狠狠扎在张三心上 | 删除整个比喻，用生理反应替代（心跳漏了一拍、手突然攥紧） |
| 🔴 字如重锤 | 每一个字/句话，都像…… | 他每一个字，都像一记重锤 | 去掉比喻，直接说「他说完那句话，全场安静了三秒」 |
| 🔴 三段式人设 | （代词），（人名），（身份/行动） | 他，张三，龙族太子，拔剑而出 | 拆成两句叙述：先动作，再补身份 |
| 🔴 万能比方 | 像……一样 / 仿佛……一般 | 像一只受伤的野兽，他蜷缩在角落 | 直接陈述事实；若必须比喻，用个人化新颖喻体 |
| 🔴 否定煽情 | 不是 X 而是 Y / 不仅是一…更是一… | 失败不是终点，而是新的起点 | 直接写事实：「他输了，回去加练了三个月」 |
| 🔴 虚假范围 | 从 X 到 Y（无实际信息增量） | 从田间到餐桌，从历史到未来 | 写具体事物，不用宏大排比 |
| 🔴 破折号泛滥 | 一段出现多个「——」 | AI 长篇中每段至少 2-3 个破折号 | 一段不超过 1 个破折号，宁用句号断开 |
| 🔴 过度排比 | 三段式以上整齐句式 | 三点并列、三段对称 | 故意打破节奏：长短句交替，穿插单句成段 |

> ⚠️ 这类句式本质是英译中（英文原著中常见），中文原创写作极少出现。全文 Ctrl+F 搜索「像」「仿佛」「不是……而是」，标记后整句改写。

## 二十·过度拔高类形容词
> **问题本质**：AI 喜欢给事物贴"史诗级"标签，人类只在真正值得时才用最高级形容。这类词让文字飘在空中，缺乏实感。

| 风险等级 | 禁用词 | 典型问题 | 替代建议 |
|:---|:---|:---|:---|
| 🔴 | 历史性 / 里程碑式 / 划时代 | 把普通进展包装成「人类文明的一大步」 | 除非真的如此，否则直接删除。比如「这是一次划时代的突破」→删掉，直接说做了什么 |
| 🔴 | 高光 / 决定性 / 关键性 | 每一件事都是「关键性的」，稀释了词的真实分量 | 替换为具体事实。「他在关键性时刻挺身而出」→「对方只剩3秒，他接了球」 |
| 🟡 | 压倒性 / 不容置喙 / 不言而喻 | 强行替读者下结论，剥夺读者自己判断的空间 | 直接说结论，不加前缀。「不言而喻，这件事是错的」→「这件事错了」 |
| 🟡 | 毋庸置疑 | 每篇AI文章至少出现一次，已完全贬值 | 删掉。如果事情真的毋庸置疑，你不需要说它毋庸置疑 |
| 🟡 | 淬了毒 / 淬了冰 / 淬了火 | 网络文套路句式，AI大量复刻后已毫无新鲜感 | 慎用，或替换为具体动作描写。「眼神淬了毒」→「她盯着他，眼都不眨」 |

> 💡 **速查口诀**：「历史性」→删；「关键性」→用事实替代；「毋庸置疑」→直接删；「淬了X」→把比喻换成动作。

## 二十一·常见但被滥用的「假 AI 词」
> **问题本质**：以下词本身正常、合理、日常也会用，但因为 AI 高频使用已被「污染」——读者看到这些词的第一反应不再是它们在说什么，而是「这是 AI 写的」。建议酌情替换或删减，尤其**频率过高时**。

### 频率检测参考

| 词类 | 成员 | 判别建议 |
|:---|:---|:---|
| 转折/承接词 | 显然 / 果然 / 突然 | 全文出现 >3 次时开始标记替换。段落开头出现时优先砍掉。 |
| 渐近词 | 渐渐 / 逐渐 / 不断 | 连续两段都出现时合并或删掉一个。「不断」「逐渐」常可省略——动作本身就隐含渐进。 |
| 内心戏开头 | 想到 / 感受 / 意识到 | AI 用这三个词作为「内心描写」的标准起手式。换成动作、对话或跳跃式联想。如「他意识到自己错了」→「他停了下来，把话咽了回去」。 |
| 程度放大器 | 巨幅 / 显著 / 极大 / 大幅 | 用具体数字替代。「显著提升」→「翻了3倍」；「大幅下降」→「从80%掉到30%」。没有数字支撑时，说明不需要这个词。 |

### 高频替换速查

| 假AI词 | 判别阈值 | 替换方向 |
|:---|:---|:---|
| 显然 | 全文 >2 次 | 直接删除，或换成「说实话」「看得出来」 |
| 果然 | 全文 >2 次 | 换成「还真是」「没跑」 |
| 突然 | 全文 >3 次 | 换动作描写替代，如「门开了」→不需要「突然」 |
| 渐渐 | 连续两段 | 删掉或换成「一点点」「慢慢」 |
| 不断 | 连续两段 | 大多时候直接删，动词本身已暗示持续 |
| 意识到 | 全文 >2 次 | 换成身体反应或动作，如「手停了一下」「话说到一半咽回去了」 |
| 显著 | 每次 | 用数据替代，没有数据就删 |
| 极大 | 每次 | 用数据替代，没有数据就删 |

> 💡 **核心原则**：这些词不是绝对不能用的「毒药」——是「红绿灯」。单次使用没问题（黄灯），频率一高就报警（红灯）。全文写完后 Ctrl+F 搜一遍，哪个词出现超过 3 次，当场削掉两个。
